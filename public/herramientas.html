Herramientas de Hacking

Nmap

Grupo: White Hat, Grey Hat, Black Hat
Función: Escaneo de redes y puertos, detección de servicios y sistemas operativos.
Descripción: Nmap es la herramienta más popular para el reconocimiento de redes. Permite descubrir hosts y servicios en una red, identificar sistemas operativos y detectar vulnerabilidades básicas.
Instalación: sudo apt install nmap (Linux) | Descargar para otros sistemas
Métodos famosos: nmap -A -T4 192.168.1.1 (detección avanzada), nmap -sS 192.168.1.0/24 (escaneo sigiloso)
Ejemplo de salida:
PORT   STATE SERVICE VERSION
22/tcp open  ssh     OpenSSH 7.6p1
80/tcp open  http    Apache httpd 2.4.29
Ventajas: Rápido, flexible, soporta scripts NSE para automatizar tareas.
Limitaciones: Puede ser detectado por firewalls y sistemas IDS/IPS.
Repositorio oficial en GitHub | Documentación oficial

Hydra

Grupo: Black Hat, Grey Hat, White Hat
Función: Ataques de fuerza bruta a servicios de red (SSH, FTP, HTTP, etc.).
Descripción: THC-Hydra es una de las herramientas más rápidas y flexibles para realizar ataques de fuerza bruta a contraseñas en múltiples protocolos.
Instalación: sudo apt install hydra (Linux) | Compilar desde GitHub
Métodos famosos: hydra -l admin -P passwords.txt 192.168.1.1 ssh (fuerza bruta SSH)
Ejemplo de salida:
[22][ssh] host: 192.168.1.1   login: admin   password: password123
Ventajas: Soporta muchos protocolos, muy rápida, personalizable.
Limitaciones: Puede ser bloqueada por sistemas de protección contra fuerza bruta.
Repositorio oficial en GitHub | Más info

SQLmap

Grupo: Black Hat, Grey Hat, White Hat
Función: Automatiza la detección y explotación de vulnerabilidades de inyección SQL.
Descripción: SQLmap es la herramienta más famosa para automatizar ataques de inyección SQL, permitiendo extraer bases de datos, usuarios y contraseñas de forma sencilla.
Instalación: sudo apt install sqlmap (Linux) | Descargar desde GitHub
Métodos famosos: sqlmap -u \"http://objetivo.com/vuln.php?id=1\" --dbs
Ejemplo de salida:
[INFO] testing connection to the target URL
[INFO] parameter 'id' is vulnerable
[INFO] fetching database names
available databases [2]:
[*] information_schema
[*] users
Ventajas: Muy automatizada, soporta múltiples DBMS, fácil de usar.
Limitaciones: Puede ser detectada por WAFs y sistemas de protección web.
Repositorio oficial en GitHub | Web oficial

Nikto

Grupo: White Hat, Grey Hat
Función: Escaneo de vulnerabilidades en servidores web.
Descripción: Nikto es un escáner de servidores web que detecta archivos peligrosos, configuraciones inseguras y vulnerabilidades conocidas.
Instalación: sudo apt install nikto (Linux) | Descargar desde GitHub
Métodos famosos: nikto -h http://objetivo.com
Ejemplo de salida:
+ Server: Apache/2.4.29 (Ubuntu)
+ /admin/: Admin login page found.
+ /phpinfo.php: PHP info file found.
Ventajas: Fácil de usar, detecta muchas vulnerabilidades conocidas.
Limitaciones: Puede generar mucho tráfico y ser detectado fácilmente.
Repositorio oficial en GitHub | Web oficial

John the Ripper

Grupo: Black Hat, Grey Hat, White Hat
Función: Cracking de contraseñas mediante diccionario o fuerza bruta.
Descripción: John the Ripper es una herramienta para descifrar contraseñas a partir de hashes, muy usada en auditorías de seguridad y CTFs.
Instalación: sudo apt install john (Linux) | Descargar desde GitHub
Métodos famosos: john --wordlist=rockyou.txt hashes.txt
Ejemplo de salida:
Loaded 1 password hash (sha512crypt)
password123      (root)
1g 0:00:00:02 DONE
Ventajas: Soporta muchos algoritmos, rápido, ampliamente documentado.
Limitaciones: Puede requerir mucha CPU y memoria para hashes complejos.
Repositorio oficial en GitHub | Web oficial
